# Week 4: Viewing

## Topics Covered

- **Views** — Virtual tables defined by queries
- **CREATE VIEW** — Defining reusable query results
- **Views for Simplifying** — Abstracting complex queries
- **Views for Aggregating** — Pre-computed summaries
- **Temporary Views** — Session-specific virtual tables
- **CREATE TEMPORARY VIEW** — Temporary query abstractions
- **Common Table Expressions (CTEs)** — WITH clauses for readable queries
- **Views for Partitioning** — Organizing large datasets
- **Views for Securing** — Controlling data access
- **Soft Deletions** — Combining views with deletion flags
