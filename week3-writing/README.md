# Week 3: Writing

## Topics Covered

- **CRUD Operations** — Create, Read, Update, Delete
- **INSERT INTO** — Adding new rows to tables
- **CSVs** — Working with comma-separated value files
- **.import** — SQLite command for importing CSV data
- **DELETE FROM** — Removing rows from tables
- **Foreign Key Constraints** — Maintaining referential integrity on deletion
- **UPDATE** — Modifying existing data
- **Triggers** — Automated actions on INSERT, UPDATE, or DELETE
  - **BEFORE** triggers
  - **AFTER** triggers
  - **INSTEAD OF** triggers
- **Soft Deletions** — Marking rows as deleted instead of removing them
