# Week 2: Designing

## Topics Covered

- **Schemas** — Defining database structure with CREATE TABLE
- **Normalizing** — Organizing data to reduce redundancy (1NF, 2NF, 3NF)
- **Data Types** — TEXT, INTEGER, REAL, NUMERIC, BLOB
- **Storage Classes** — SQLite's type affinity system
- **Type Affinities** — How SQLite handles type conversion
- **Table Constraints**
  - **PRIMARY KEY** — Unique row identifier
  - **FOREIGN KEY** — Relationship to another table
- **Column Constraints**
  - **CHECK** — Custom validation rules
  - **DEFAULT** — Default values for columns
  - **NOT NULL** — Requiring values
  - **UNIQUE** — Ensuring uniqueness
- **Altering Tables**
  - **DROP TABLE** — Removing tables
  - **ALTER TABLE** — Modifying existing tables
  - **ADD COLUMN** — Adding new columns
  - **RENAME COLUMN** — Renaming columns
  - **DROP COLUMN** — Removing columns
