# Week 0: Querying

## Topics Covered

- **Tables, Databases, and DBMSs** — Understanding relational database fundamentals
- **SQLite** — Introduction to embedded database system
- **SELECT** — Basic data retrieval from tables
- **LIMIT and OFFSET** — Controlling result set size and pagination
- **WHERE** — Filtering rows based on conditions
- **Comparisons** — =, !=, <, >, <=, >= operators
- **NOT** — Negating conditions
- **NULL** — Handling missing or unknown values
- **Pattern Matching with LIKE** — Text pattern searches with wildcards (%, _)
- **Compound Conditions** — AND, OR logical operators
- **Range Conditions** — BETWEEN, IN operators
- **ORDER BY** — Sorting results (ASC, DESC)
- **Aggregate Functions** — COUNT, SUM, AVG, MIN, MAX
- **ROUND** — Rounding numerical results
- **DISTINCT** — Eliminating duplicate rows
